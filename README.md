1. Install Node.js 10+.
1. [Create a token.](https://github.com/settings/tokens/new?description=archive%20script&scopes=repo)
1. Clone this Gist.
1. From this directory, install the dependencies.

   ```sh
   npm install
   ```

1. Run the script.

   ```sh
   ORG=... GITHUB_TOKEN=... node archive.js
   ```
